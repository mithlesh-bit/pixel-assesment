Domain Cart UI

Overview

Domain Cart UI is a web application that allows users to manage their domain shopping cart efficiently. Users can check domain availability, remove unavailable domains, copy domain names to the clipboard, and keep the best domains based on predefined criteria.

Features

Add Domains: Users can add domains to the cart.

Check Availability: Automatically checks domain availability and updates the UI.

Remove Unavailable Domains: Clears out domains that are not available.

Clear Cart: Removes all domains from the cart.

Copy Domains: Copies selected domain names to the clipboard.

Keep Best Domains: Filters and retains the best domains based on domain extension priority.

Technologies Used

React (Next.js): Frontend framework for rendering UI.

TypeScript: Ensures type safety and better maintainability.

Lucide-react: Provides icons for UI elements.

SweetAlert2: Used for displaying popup alerts.

Local Storage: Stores domain availability status.

Installation

Clone the repository:

git clone https://github.com/yourusername/domain-cart-ui.git

Navigate to the project folder:

cd domain-cart-ui

Install dependencies:

npm install

Run the development server:

npm run dev

Open your browser and go to http://localhost:3000

Usage

Add domains to your cart.

Click "Remove Unavailable" to delete unavailable domains.

Click "Clear Cart" to empty the cart.

Click "Copy Domains" to copy domain names.

Click "Keep Best Domains" to retain the top-ranked domains.
